DSP空间化迭代方案：

机器按人工设定的算法域混音 to 人工打分+机械学习 = 规则自动化混音

可以这么理解：这是一个**机械学习/规则学习系统**，但还不是通常意义上的 AI 模型。

更准确地说，它会是：

**rule-based adaptive system + feedback controller + preference memory**

也就是：

```text
分析歌曲特征
-> 生成初始空间化参数
-> 渲染一次
-> 测量结果
-> 根据规则自动修正
-> 记录你的听感反馈
-> 下次用反馈修正规则权重
```

它“学习”的不是神经网络权重，而是这些东西：

- 哪些指标代表“后方不明显”
- 哪些指标代表“低频被吃掉”
- 哪些参数能补回来
- 你的偏好是更大胆还是更保守
- 哪类歌应该自动更宽、更厚、更亮或更克制

所以它很像传统控制系统里的：

- 自适应滤波
- 闭环控制
- 专家系统
- 参数寻优
- 用户偏好校准

不是黑箱 AI，而是**可解释的机械学习**。

区别大概是：

```text
现在：
我觉得 bass 少 -> 手动告诉你 -> 改规则

升级后：
系统发现 bass retention 低 -> 自动补 bass_gain -> 保存这次修正

再升级：
你多次说“还是不够突出” -> 系统提高你的默认 spatial_intensity / bass_preference
```

这其实很适合你的场景。因为音乐空间化不是一个纯客观问题，里面有大量“听感偏好”。完全固定规则会死，纯 AI 又不可控。机械学习刚好在中间：**能自动适应，但每一步都能解释、能回滚、能调。**
